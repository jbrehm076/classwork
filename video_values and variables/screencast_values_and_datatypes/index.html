Values and Datatypes

  • Numbers

    Literally just the number. ex: 2, 30, 23343, 101101001001

  • Strings

    Stores a sequence of characters. Envision data encapsulation between OSI layers in networking. Quotation marks encase string values!

  • Boolean

    These have on and off, or true and false values. Digital functionality i.e. boolean algebra for IC circuits.
